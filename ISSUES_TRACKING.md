## Creating New Issues

Use the GitHub web interface or CLI:

```bash
# Create a new issue
gh issue create --title "Issue title" --body "Issue description"

# List all issues
gh issue list

# View a specific issue
gh issue view <issue-number>
```

## Scripts

- **create_issues.sh** - Batch create issues from a list
- **sync_todos_to_issues.sh** - Sync TODO items with GitHub issues
