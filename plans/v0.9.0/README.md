# Version 0.9.0 - Long Rides Enhancement

**Release Date:** March 30, 2026  
**Original Tag:** v2.4.0 (renumbered to v0.9.0)  
**Architecture:** Flask + SQLAlchemy + Enhanced Long Rides

## Overview

This release enhanced the long rides feature with comprehensive statistics, interactive maps, and improved user experience. It also included template refactoring to reduce technical debt.

## Key Features

- **Long Rides Statistics** - Top 10 longest rides with Strava links
- **Monthly Breakdown** - Monthly ride statistics
- **Performance Metrics** - Average speed and elevation gain
- **Interactive Maps** - Visual display of all long ride routes
- **Template Refactoring** - Reduced technical debt in templates
- **Geocoding Completion** - Background geocoding fully implemented
