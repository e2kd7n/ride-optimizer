# Version 0.6.0 - Flask + SQLAlchemy + Design System

**Release Date:** March 26, 2026  
**Original Tag:** v2.1.0 (renumbered to v0.6.0)  
**Architecture:** Flask + SQLAlchemy + Web Interface

## Overview

This release marked the transition from CLI to a web-based architecture using Flask and SQLAlchemy. It introduced a comprehensive design system and focused on code quality improvements.

## Key Features

- **Flask Web Framework** - Transition from CLI to web-based interface
- **SQLAlchemy ORM** - Database abstraction layer
- **Design System** - 10 core design principles established
- **Improved Exception Handling** - Better error management
- **SHA256 Security Migration** - Enhanced security measures
- **UI/UX Redesign Roadmap** - Epic #62 with 7 sub-issues
